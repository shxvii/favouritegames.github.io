Boomerang Kid

This pixel-art game stars a boy, named as “boomerang kid “who wants to defeat a boss up the treacherous mountain, with a boomerang of course. But to do so, he must dodge the attacks of a canon, go over spikes and avoid slimes. It has stunning art and a fun game play.

Bionic Blitz

Bionic Blitz is a pixel-based game in which the protagonist must shoot all the bots with a purple laser beam, but you can’t shoot for too long as your battery (the purple bar above your head) runs out and when it does it takes about a few seconds to reload. There are randomly generated waves of robots. This is an entertaining game and catchy soundtrack.

Everest

This game is all about a restless hiker (you) trying to reach the top of Mount Everest. You must pass all the different challenges to reach and what makes this game harder is that there is no respawn so once you die you must start over. This is an almost relaxing game with beautiful graphics and realistic gameplay.
